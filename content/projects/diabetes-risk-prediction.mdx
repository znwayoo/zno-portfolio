# Diabetes Risk Prediction: From Data Pipeline to Production App

Built a multi-stage data pipeline processing 400K+ CDC BRFSS records through data cleaning, feature engineering, statistical analysis, and predictive modeling, then deployed the results as two client-facing applications.

## The Pipeline

Starting from 433,323 records with 350 variables, I reduced the feature space to 14 key predictors through systematic feature selection, removing columns with 60-95% missing values and applying domain-driven feature engineering.

### Data Processing
- Cleaned and standardized raw CDC BRFSS 2023 survey data
- Applied SMOTE for class imbalance handling
- Used StandardScaler for feature normalization

### Model Training & Evaluation

Trained and evaluated 6 model types:

| Model | Accuracy | Recall |
|-------|----------|--------|
| KNN | **0.84** | **0.93** |
| XGBoost | 0.83 | 0.89 |
| LightGBM | 0.83 | 0.88 |
| Random Forest | 0.82 | 0.87 |
| Logistic Regression | 0.81 | 0.86 |
| Decision Tree | 0.78 | 0.82 |

### Model Interpretability

Applied SHAP (SHapley Additive exPlanations) for model interpretability. Key finding: **high blood pressure is the strongest predictor** of diabetes risk.
